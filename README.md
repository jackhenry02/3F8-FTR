# 3F8--FTR

## Overview
This is the 3F8--FTR project.

## Installation
To set up this project, run:
```bash
git clone https://github.com/jackhenry02/3F8--FTR.git
cd 3F8--FTR
python -m venv venv
source venv/bin/activate  # For Mac/Linux
pip install -r requirements.txt
```

## Usage
Run the main program:
```bash
python src/main.py
```
